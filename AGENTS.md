# AGENTS

Project conventions for automation and helper scripts.

## Layout

- `src/` - core package code (`chess_tui`)
- `scripts/` - runnable helper scripts
- `tests/` - unit tests
- `docs/` - project documentation

## Scripts

Use `source ./activate` to enter the virtual environment.

Common commands:

- `chess-tui` - run the package CLI
- `check-setup` - verify the environment and package import
- `update-deps` - install or update the project and development dependencies
- `fix-deps` - compatibility alias for `update-deps`
